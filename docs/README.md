## 구현할 기능 목록

1. 랜덤 숫자 생성 기능

   - 1부터 9사이의 서로 다른 수 3개로 이루어짐

2. 사용자 숫자 입력 기능

   - 잘못된 입력값 처리 기능
     - alert으로 에러 메시지 띄움
     - 다시 입력이 가능하게 함

3. 랜덤 숫자와 사용자 숫자 비교 기능

   - 3개의 숫자를 모두 맞히면 게임 종료
   - 맞추지 못한 경우 힌트 제공

4. 결과 출력 기능

   - 오답 시 힌트 제공 기능
     - 같은 수가 같은 자리에 있으면 `스트라이크`
     - 같은 수가 다른 자리에 있으면 `볼`
     - 같은 수가 전혀 없으면 `낫싱`
   - 정답 시 재시작 버튼 노출 기능
